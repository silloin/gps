# Pitching RunTerra as a Final Year Project

## 1. Project Title

**RunTerra: A Real-Time, GPS-Based Territory Capture Game**

## 2. Project Overview

RunTerra is a full-stack MERN web application that combines fitness and gaming. It allows users to capture and defend territory on a real-world map by running. The project leverages modern web technologies to create a live, interactive, and competitive experience for all players.

## 3. Key Features and Technical Highlights

*   **Real-Time GPS Tracking and Mapping:** Utilizes the Geolocation API and Mapbox GL to track users' routes and display them on a live map.
*   **Territory Capture and Defense:** Implements a grid-based system for capturing and stealing territory, with real-time updates broadcasted to all users via Socket.io.
*   **King/Queen System:** Groups tiles into zones (neighborhoods, cities) and crowns the user with the most territory as the "King" or "Queen" of that zone.
*   **Comprehensive Running and Training Features:** Includes a stats dashboard, personalized training plans, and an interval workout mode.
*   **Competition and Community:** Features global and local leaderboards, challenges, events, and Strava integration to foster a sense of community and competition.
*   **Modern Tech Stack:** Built with the MERN stack (MongoDB, Express, React, Node.js), ensuring a robust and scalable application.

## 4. Innovation and Originality

*   **Gamification of Fitness:** RunTerra offers a unique and engaging way to motivate people to run and stay active.
*   **Real-Time, Global Gameplay:** The use of Socket.io to create a single, global game world is a key innovation.
*   **Community-Driven Competition:** The King/Queen system and local leaderboards encourage friendly competition and social interaction.

## 5. Project Goals and Objectives

*   To design and develop a full-stack web application using the MERN stack.
*   To implement real-time features using Socket.io.
*   To integrate with third-party APIs such as Mapbox and Strava.
*   To create a fun and engaging user experience that encourages physical activity.
*   To deploy the application to a cloud-based platform (e.g., Vercel, Render).
